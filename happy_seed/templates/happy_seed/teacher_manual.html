📘 행복의 씨앗 교사용 상세 설명서

긍정 행동 기록부터 꽃피움 운영까지, 수업 중 바로 쓰는 실전 가이드

1. 수업 시작 전 준비

  • 교실 생성 후 학생을 등록합니다.
  • 동의 관리에서 학부모 명단만 입력/업로드하면 consent 링크와 안내문이 자동 생성됩니다.
  • 보상 관리에서 보상별 선택 확률(%)과 수량을 입력합니다.
  • 설정 화면에서 기본 당첨 확률, 씨앗 전환 기준(N)을 확인합니다.

2. 수업 중 운영 순서

  1. 성실 참여 학생에게 꽃피움 티켓을 지급합니다.
  2. 우수 성취 학생에게 추가 티켓을 지급합니다.
  3. 꽃피움 실행 후 미당첨이면 씨앗이 자동 적립됩니다.
  4. 씨앗이 N개가 되면 티켓으로 자동 전환됩니다.
  5. 축하 화면은 교사가 직접 닫아 공동체 멘트를 마무리합니다.

3. 보상 확률 설정 원칙

보상별 선택 확률(%)은 당첨이 발생했을 때 어떤 보상이 선택될지의 상대 비율입니다.

  • 예시: 역할 보상 70, 선택권 보상 20, 물질 보상 10
  • 합계가 100이 아니어도 상대 비율로 계산됩니다.
  • 보상 확률은 학생에게 공개하지 않는 것을 권장합니다.

4. 동의 운영(consent 자동 생성)

  • 동의 관리 화면에서 학생명/학부모명 명단을 붙여넣거나 CSV로 업로드합니다.
  • 동의 링크 자동 생성 버튼을 누르면 안내문 PDF와 학생별 링크가 자동 생성됩니다.
  • 학부모 응답 후 consent 결과 동기화를 눌러 상태를 반영합니다.

나의 작은 행동 하나하나가 나의 미래, 너의 미래, 우리 모두의 미래를 행복으로 바꿉니다.
