## 121. 반짝반짝 우리반 알림판

### 1. 서비스 한눈에 보기

- 서비스명: 반짝반짝 우리반 알림판
- 진입 경로: /products/dutyticker/
- 라우트 이름: dutyticker
- 핵심 기능: 한눈에 보는 역할표, 스스로 챙기는 책임감, 활동 집중 타이머
- 설명: 선생님, 저 지금 뭐해요? 우리 반 아이들이 할 일을 잊지 않도록 화면으로 챙겨주세요.

### 2. 시작 전 준비

1. 수업 목표(무엇을 수집/생성/운영할지)를 먼저 정합니다.
2. 필요한 입력 자료(명단/문서/주제/파일)를 사전에 준비합니다.
3. 대화면 운영 시 PC에서 먼저 테스트 후 공유 링크를 배포합니다.

### 3. PC 운영 동선

1. 상단 `설정(톱니)`에서 학생 명단/역할/시간표를 먼저 저장합니다.
2. 메인 화면 중앙 타이머 숫자를 눌러 시작/일시정지하고, 하단 `+1분`, `5분`, `10분`으로 조정합니다.
3. 우측 `오늘의 역할`에서 `역할 순환`과 `호명 모드`를 활용해 발표 흐름을 만듭니다.
4. 우측 알림 카드 `펜` 버튼으로 공지를 입력해 즉시 반영합니다.

### 4. 모바일 운영 동선

- 모바일은 명단/역할 수정 중심으로 사용하고, 메인 노출은 PC/대화면을 권장합니다.
- 공지는 짧은 문장(줄바꿈 포함)으로 작성하면 학생 화면 가독성이 좋아집니다.

### 5. 수업 적용 예시

1. 수업 시작 전 `반짝반짝 우리반 알림판` 화면을 열어 오늘 활동 목표를 안내합니다.
2. 활동 중간에 입력/응답 상태를 점검하고 누락 대상을 즉시 안내합니다.
3. 수업 종료 직전에 `알림판 공지 카드`을(를) 저장·공유해 마무리합니다.

### 6. 자주 묻는 문제와 해결

- 실행 속도가 느리면 입력 데이터를 줄여서 1차 테스트(권장: 2분)를 먼저 진행합니다.
- 결과가 기대와 다르면 핵심 입력값(제목/옵션/대상/형식)을 다시 확인합니다.
- 공유가 안 될 때는 링크 권한, 만료 상태, 브라우저 팝업 차단 여부를 점검합니다.
