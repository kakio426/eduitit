## 121. 반짝반짝 우리반 알림판

- 카테고리: 수업·학급 운영
- 이용 조건: 로그인 필요
- 접근 방식: Eduitit 내부 경로 `/products/dutyticker/`
- 해결 문제: 학급 소식을 빠르게 공유해요
- 기대 결과: 알림판 공지 카드
- 예상 시간: 2분

### 1. 서비스 한눈에 보기
- 배지: 핵심

- 서비스명: **반짝반짝 우리반 알림판**
- 카테고리: **수업·학급 운영**
- 접근 방식: Eduitit 내부 경로 `/products/dutyticker/`
- 이용 조건: **로그인 필요**
- 해결 문제: 학급 소식을 빠르게 공유해요
- 기대 결과: 알림판 공지 카드
- 예상 시간: 2분

### 2. 시작 전 준비
- 배지: 준비

1. 사용할 학급/대상/기간 정보를 먼저 정리합니다.
2. 공유가 필요한 경우 링크 공개 범위(보기/편집)를 미리 결정합니다.
3. 결과를 어디에 기록/배포할지(알림장, 가정통신문, 내부 문서) 미리 정합니다.
4. 개인정보가 포함될 수 있으면 익명화 또는 최소 입력 원칙을 적용합니다.

### 3. 단계별 사용 방법
- 배지: 절차

1. Eduitit 홈에서 `반짝반짝 우리반 알림판`를 검색해 진입합니다. (Eduitit 내부 경로 `/products/dutyticker/`)
2. 첫 화면에서 학급/대상/기간 등 기본 범위를 먼저 정합니다.
3. 핵심 입력값(문항, 안내문, 명단, 조건)을 넣고 실행/생성 버튼을 누릅니다.
4. 결과 화면에서 누락·오류를 먼저 확인한 뒤 저장 또는 공유합니다.
5. 수업/업무 적용 후 필요한 항목을 수정하고 최종본을 다시 배포합니다.

### 4. 핵심 기능 1. 한눈에 보는 역할표
- 배지: 기능

### 기능 설명
현재 누가 어떤 역할을 맡고 있는지 큼직한 화면으로 보여주어 아이들이 잊지 않습니다.

### 실제 사용 순서
1. `한눈에 보는 역할표`와 관련된 메뉴를 선택합니다.
2. 필요한 입력값을 확인하고 기본값이 있으면 먼저 검토합니다.
3. 실행 후 생성된 결과를 확인하고 학급/업무 맥락에 맞게 수정합니다.
4. 확정된 결과를 저장·공유·출력 중 필요한 방식으로 마무리합니다.

### 현장 적용 팁
- 기능 단위로 결과를 검토하고 다음 단계로 넘어가면 오류 누적을 줄일 수 있습니다.
- 반복 사용 시 입력 템플릿(문구/명단/조건)을 미리 준비해 두면 시간이 크게 단축됩니다.

### 2. 시작 전 준비
- 배지: 준비

1. 사용할 학급/대상/기간 정보를 먼저 정리합니다.
2. 공유가 필요한 경우 링크 공개 범위(보기/편집)를 미리 결정합니다.
3. 결과를 어디에 기록/배포할지(알림장, 가정통신문, 내부 문서) 미리 정합니다.
4. 개인정보가 포함될 수 있으면 익명화 또는 최소 입력 원칙을 적용합니다.

### 3. 단계별 사용 방법
- 배지: 절차

1. Eduitit 홈에서 `배부 체크`를 검색해 진입합니다. (Eduitit 내부 경로 `/handoff/`)
2. 첫 화면에서 학급/대상/기간 등 기본 범위를 먼저 정합니다.
3. 핵심 입력값(문항, 안내문, 명단, 조건)을 넣고 실행/생성 버튼을 누릅니다.
4. 결과 화면에서 누락·오류를 먼저 확인한 뒤 저장 또는 공유합니다.
5. 수업/업무 적용 후 필요한 항목을 수정하고 최종본을 다시 배포합니다.
