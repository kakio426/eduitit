# FortuneSheet 기반 학교 시간표 허브 v1

## 현재 문제 목록

1. 현재 `timetable` 앱은 교사가 직접 배치하는 편집 허브가 아니라 엑셀 업로드 보조 도구에 가깝다.
2. 전담 교사 겹침, 강사 더블부킹, 특별실 수용량 초과를 한 화면에서 즉시 확인하기 어렵다.
3. 회의 중 여러 반의 빈칸을 빠르게 비교할 조율 화면이 없다.
4. 교사용 확인 화면이 별도 읽기 전용 구조로 정리되어 있지 않다.
5. 향후 `reservations`와 `classcalendar`에 연결할 발행 계약이 아직 없다.

## 화면 통합안

- 관리자 허브
  - FortuneSheet 편집기를 중심으로 두고, 좌우 패널에 교사/특별실/경고/시수 정보를 붙인다.
- 회의 조율 화면
  - 반 x 시간칸 매트릭스로만 보여 주고, 녹색/빨간색/파란색 의미를 고정한다.
- 교사용 읽기 화면
  - 반 또는 교사 한 가지 관점만 보여 주고 인쇄에 맞춘다.
- 레거시 가져오기
  - 기존 엑셀 업로드 도구는 별도 경로로 내리고, 메인 첫 화면의 주 행동은 시간표 편집으로 고정한다.

## 텍스트 와이어프레임

```text
[상단]
학교 / 학년도 / 학기 / 학년
시간표 편집

[좌측]
교사 추가
교사 목록
특별실 수용량
실시간 시수

[중앙]
FortuneSheet
- 반별 시트 탭
- 요일 x 교시

[우측]
빠른 입력
충돌 경고
스냅샷 저장
확정 발행
공유 링크
```

```text
[회의 조율]
학년 선택
강사 선택
과목 / 특별실 선택

반 x 시간칸 매트릭스
- 녹색: 배정 가능
- 빨간색: 배정 불가
- 파란색: 현재 강사 배정
```

```text
[읽기 전용]
3학년 2반 주간 시간표
또는
홍길동 교사 주간 시간표

인쇄
```
