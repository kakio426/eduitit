# Eduitit Home + Catalog Teacher-First Checklist

작성일: 2026-03-08
범위: `core/home_authenticated_v2`, `products/list`, 전역 서비스 진입 구조

## 상태

- 현재 baseline 기준: `home_authenticated_v2.html`, `products/list.html`, `core/views.py`, `products/views.py`
- 이 문서는 home/catalog baseline의 same-source-of-truth checklist로 사용한다.

## 목표

- 홈은 `교사 오늘의 작업 공간`으로 보이게 한다.
- 서비스 목록은 `쇼케이스`가 아니라 `작업형 카탈로그`로 바꾼다.
- 설명, 보조 기능, 희소 기능은 기본 화면에서 줄인다.

## 최종 와이어프레임

### 1. 로그인 홈

```text
[서비스 검색........................] [전체 서비스] [이용방법]

[오늘 바로 시작]
[오늘 일정 보기] [작업 바로 시작]
-----------------------------------
오늘 일정 / 최근 작업 / 지금 처리할 일

[자주 여는 도구]
[도구 카드 4개 이내]

[핵심 업무]
수합·서명
문서·작성
학급 운영
학부모 소통

[더 많은 도구] (접힘)
가이드·인사이트
교실 활동
외부 서비스
```

원칙:
- `SNS`, `개인 캘린더`, `보조 도구`, `게임 배너`는 기본 화면에서 전면 노출하지 않는다.
- 첫 화면에서 `지금 해야 할 일`이 바로 보여야 한다.

### 2. 전체 서비스

```text
[전체 서비스]
교실에서 자주 쓰는 일을 기준으로 정리했습니다.
[홈으로] [이용방법]

[핵심 업무]
수합·서명 / 문서·작성 / 학급 운영 / 학부모 소통

[교실 활동]
게임/활동 도구

[도움말·외부]
가이드·인사이트 / 외부 서비스
```

원칙:
- 영어 카피, 가격 중심, 쇼케이스 카드 제거
- 각 카드에서 `무엇을 도와주는지`가 먼저 보여야 한다.
- 같은 서비스는 `열기`만 1차 CTA로 둔다.

### 홈

- [x] `개인 캘린더` 패널 제거

- [x] `교실 활동`을 접힘 영역으로 이동
- [x] 상단에 `전체 서비스`, `이용방법` CTA 추가
- [x] `클래스룸 워크스페이스` 카피를 `오늘 바로 시작` 기준으로 축소
- [x] `자주 여는 도구`를 4개 이내로 유지

### 카탈로그

## 완료 기준

- 홈 첫 화면에서 핵심 업무 진입과 최근 작업이 바로 보인다.
- 카탈로그가 제품 전시가 아니라 교사 업무별 목록으로 읽힌다.
- 설명을 읽지 않아도 어디로 들어가야 하는지 이해된다.
