# ArtClass Launcher Rollout Plan

## Scope
비임베드 유튜브 대응용 교사용 런처를 단계적으로 배포한다.

## Phase 1: Internal QA (1~2일)
1. 사내 Windows 11 2대에서 설치/실행 검증
2. `런처로 수업 시작` 1클릭 동작 검증
3. fallback(`화면 분할 시작`) 검증

## Phase 2: Pilot Teachers (3~5일)
1. 파일럿 교사 3명 선정
2. 실제 수업 1회 이상 사용
3. 수집 지표:
- 런처 실행 성공률
- 수업 시작까지 소요 시간
- 수동 창 조절 발생 횟수
- 런처 미설치/프로토콜 차단 건수

## Phase 3: School-wide Release
1. 설치 가이드 배포
2. 교사 공지: 기존 브라우저 분할은 fallback, 기본은 런처 사용
3. 1주 모니터링 후 기본 운영 플로우 고정

## Rollback Criteria
아래 중 하나 충족 시 즉시 fallback 우선 운영:
1. 런처 시작 실패율 5% 초과
2. 수업 시작 지연(30초 초과) 빈발
3. 특정 학교 PC 정책으로 프로토콜 실행 차단

## Recovery
1. `런처로 수업 시작` 버튼 임시 숨김(기능 플래그)
2. `화면 분할 시작` 기본 안내로 전환
3. 차단 원인(백신/권한/브라우저 정책) 수정 후 재배포
