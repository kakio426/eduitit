# Seed Quiz 성장 실행 계획 (1~4단계)

## 목표
- 교사가 만든 문제를 자산으로 계속 쌓이게 만들고,
- 다시 쓰기 쉬운 구조를 만들어 자발적 참여를 늘린다.

## 1단계 (지금 시작): 세트 영구 저장 + 보관함/필터
- [x] 생성/선택 시 기존 draft 덮어쓰기 제거, 항상 새 세트 생성
- [x] CSV/붙여넣기 저장 시 기존 세트 갱신 대신 새 세트로 저장
- [x] 교사용 `내 문제 보관함` 화면 추가
- [x] 보관함 필터: 주제/학년/난이도/기간/검색/정렬
- [x] 보관함 조건 기반 랜덤 1세트 선택
- [ ] 운영 데이터 점검(재사용률, 생성→배포 시간)

## 2단계: 자동 공유 + 기본 운영 제어
- [x] 생성 즉시 공유(공개+승인) 기본값 적용
- [x] 검토대기 UI 제거(교사 화면 단순화)
- [ ] 서비스별 공유 ON/OFF 정책(기본 ON) 정리

## 3단계: 중복 경고 + 기본 품질검사
- [ ] 중복 경고(제목/문항 유사도)
- [ ] 형식 검사(필수값/정답범위/보기 중복)
- [ ] 금칙어/기본 문장 품질 검사
- [ ] 교사에게 쉬운 오류 문구 제공

## 4단계: 채택/즐겨찾기/신고
- [ ] 채택/즐겨찾기 버튼
- [ ] 신고 접수/처리 플로우
- [ ] 악용 방지 룰(자기 채택 제외 등)

## 관측 지표
- 생성→배포 완료 시간
- 보관함 재사용률
- 공유 세트 사용률
- 신고율
