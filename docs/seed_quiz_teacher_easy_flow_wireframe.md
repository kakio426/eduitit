# 씨앗 퀴즈: 교사용 쉬운 사용 안내 + 와이어프레임

## 1) 교사 안내 문구(쉬운 버전)

### 한 줄 안내
- "문제 만들기부터 학생 풀이까지, 3단계면 끝납니다."

### 단계별 안내
1. 문제 만들기  
- 문제를 직접 입력하거나, 다른 곳에서 만든 문제를 붙여넣습니다.

2. 미리보기 확인  
- 문제 수, 학년, 정답 번호가 맞는지 화면에서 바로 확인합니다.

3. 학생에게 풀게 하기  
- "배포하기"를 누르면 학생용 문제가 열립니다.  
- 학생들은 이름을 입력하고 바로 풉니다.

### 교사용 주의 문구(쉬운 버전)
- 한 번에 올리는 문제는 같은 학년, 같은 주제로 맞춰 주세요.
- 보기 4개(보기1~보기4)는 모두 채워 주세요.
- 정답 번호는 1, 2, 3, 4 중 하나만 써 주세요.
- 막히면 "샘플 파일"을 내려받아 그대로 따라 쓰면 됩니다.

---

## 2) 교사 화면 와이어프레임(간단 구조)

### A. 교사 대시보드 메인
```text
+------------------------------------------------------+
| [씨앗 퀴즈]                                          |
| 오늘 수업 문제를 빠르게 준비하세요.                 |
+------------------------------------------------------+
| [문제 붙여넣기] [CSV 업로드] [샘플 파일 받기]       |
| [제작 가이드 보기]                                   |
+------------------------------------------------------+
| 최근 만든 문제 세트                                  |
| - 3학년 맞춤법 20문항                                |
| - 5학년 단위 변환 10문항                              |
+------------------------------------------------------+
```

### B. 문제 붙여넣기 화면
```text
+------------------------------------------------------+
| 문제 붙여넣기                                        |
|                                                      |
| [여기에 문제를 붙여넣으세요...]                      |
|                                                      |
| (지원 형식: 표 형태 붙여넣기, 줄글 붙여넣기)        |
|                                                      |
| [붙여넣기 미리보기]                                  |
+------------------------------------------------------+
| 안내                                                  |
| - 한 번에 올리는 문제는 같은 학년/같은 주제로       |
| - 정답 번호는 1~4                                    |
+------------------------------------------------------+
```

### C. 미리보기/검수 화면
```text
+------------------------------------------------------+
| 미리보기 완료                                        |
| 총 20문항 / 3학년 / 맞춤법                           |
+------------------------------------------------------+
| 1번. 다음 중 ...                                     |
| 보기1 ... 보기2 ... 보기3 ... 보기4 ...              |
| 정답: 2                                               |
| 해설: ...                                             |
| [수정]                                                |
+------------------------------------------------------+
| [확인 후 저장] [다시 붙여넣기]                       |
+------------------------------------------------------+
```

### D. 배포(학생에게 풀게 하기) 화면
```text
+------------------------------------------------------+
| 문제 배포                                             |
|                                                      |
| 세트명: 3학년 맞춤법 20문항                           |
| 대상: 우리반                                          |
|                                                      |
| [배포하기]                                            |
+------------------------------------------------------+
| 배포 후 안내                                          |
| - 학생용 입장 링크 복사                               |
| - QR 보기                                             |
+------------------------------------------------------+
```

### E. 학생 화면(최소 단계)
```text
+------------------------------------------------------+
| 오늘의 퀴즈                                           |
| 이름 [        ]  번호 [   ]                           |
| [시작하기]                                            |
+------------------------------------------------------+
| 1번 문제                                              |
| ( ) 보기1  ( ) 보기2  ( ) 보기3  ( ) 보기4            |
| [다음]                                                |
+------------------------------------------------------+
| 결과                                                  |
| 점수: 18/20                                           |
| 틀린 문제 해설 보기                                   |
+------------------------------------------------------+
```

## 3) 교사가 헷갈리지 않게 하는 버튼/문구 제안

### 버튼 이름(쉬운 말)
- "붙여넣기 미리보기"
- "확인 후 저장"
- "학생에게 배포하기"
- "링크 복사"
- "QR 보기"

### 오류 문구(쉬운 말)
- "문제가 비어 있습니다. 내용을 붙여넣어 주세요."
- "정답 번호는 1~4만 입력할 수 있어요."
- "한 번에 올리는 문제는 같은 학년으로 맞춰 주세요."
- "보기는 4개를 모두 채워 주세요."

---

## 4) 실제 수업용 빠른 동선(교사 기준 1분)

1. 문제 붙여넣기  
2. 미리보기에서 2~3문항만 빠르게 확인  
3. "확인 후 저장"  
4. "학생에게 배포하기"  
5. 링크/QR로 학생 입장
