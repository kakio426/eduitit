Gemini API Key가 무엇인가요?

👨‍🏫 현직 선생님이 직접 만들고 운영하는 서비스입니다.

AI API 비용을 개인이 전액 감당하기가 쉽지 않아, 각자 구글에서 발급받은 열쇠(API Key)를 꽂아 사용하는 방식을 선택했습니다.

잠깐! 비용이 발생하나요?

구글의 Gemini API는 개인이 사용하기에 충분히 많은 양을 매월 무료로 제공합니다.

무엇보다 구글에 따로 카드 정보를 등록하지 않는 한, 사용량이 초과되어도 유료로 자동 전환되거나 결제될 일이 절대 없으니 안심하고 사용하세요!

개인 API 키를 등록하시면 더 빠르고 쾌적하게 AI 기능을 즐기실 수 있습니다.

* 마스터 키 사용 시 개발자의 비용 보호를 위해 사용량이 제한될 수 있습니다.

Gemini API Key 가이드

  • 1

    Google AI Studio에 로그인합니다.

Padlet API Key 가이드(선택)

  • 1

    Padlet Developer 설정에 접속합니다. (Padlet Platinum/Pro 유료 플랜 전용)

  • 2

    'Generate API Key'를 클릭하여 키를 생성합니다.

  • 3

    생성된 키를 복사하여 왼쪽 '개인 Padlet API Key' 칸에 저장하세요!
